# Angular **1.5** starter with Webpack

## What this is...
This is just a template project to start developing using Angular 1.5 and Webpack.


## Instructions
```
1) git clone https://github.com/mptorrejon/ng-webpack-starter
2) npm install
3) npm start
4) open index.html file
```
